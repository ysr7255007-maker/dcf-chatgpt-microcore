# DCF Chrome `1.0.0-rc.2` delivery

## Candidate acceptance

1. Reload or install this candidate once for browser acceptance.
2. Keep Chrome's “Allow User Scripts” enabled.
3. Open ChatGPT.
4. Confirm the unified DCF shell shows the independent function panels.
5. Confirm DCF Next or rc.1 language ammo is present; an empty library should restore from the fixed GitHub file.
6. Exercise ammo insertion/sending, long-conversation mode, appearance, backup and update check.
7. Confirm dark static pages are readable.

This is the final candidate-specific local install. Formal base releases are intended to arrive through the non-public Chrome Web Store channel rather than repeated local loading.

- candidate plugin index: candidate branch;
- formal plugin index: `main`;
- language ammo library: `language-ammo-library/data/language-ammo/library.json`;
- base release: GitHub Actions → non-public Chrome Web Store.

No real browser or store-publication acceptance is claimed by automated tests alone.
